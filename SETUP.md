# PhillyCulture — Setup & Deployment Guide

## Quick Start (Local)

1. **Clone the repo:**
   ```bash
   git clone https://github.com/eugendimant/software-cultural-events-philly.git
   cd software-cultural-events-philly
   ```

2. **Run the scraper to fetch live events:**
   ```bash
   pip install -r scraper/requirements.txt
   python scraper/scrape_events.py
   ```
   This fetches events from 14 Philadelphia cultural venues and writes `data/events.json`.

3. **Open locally:**
   ```bash
   # Any local server works. For example:
   python -m http.server 8000
   # Then open http://localhost:8000
   ```

## Deploy to GitHub Pages (Recommended)

1. **Push to GitHub** (already done if you're reading this).

2. **Enable GitHub Pages:**
   - Go to **Settings → Pages** in your repository
   - Under "Build and deployment", select **GitHub Actions**
   - The included `.github/workflows/pages.yml` will auto-deploy on push to `main`

3. **Enable the scraper workflow:**
   - Go to **Actions** tab → "Scrape Philadelphia Cultural Events"
   - Click "Enable workflow" if needed
   - The workflow runs **daily at 8 AM ET** and can also be triggered manually via "Run workflow"

4. **Your site is live at:**
   ```
   https://eugendimant.github.io/software-cultural-events-philly/
   ```

## How It Works

### Architecture
```
index.html          ← Single-page app (no build step needed)
assets/style.css    ← Dark-mode modern UI
assets/app.js       ← Client-side filtering, calendar, search
data/events.json    ← Event data (generated by scraper)
data/sources.json   ← Source metadata
scraper/            ← Python scraper for 14 venues
.github/workflows/  ← Auto-scrape + auto-deploy
```

### Features
- **List view** with color-coded category badges and month grouping
- **Calendar view** with events spanning multiple days
- **Filter by category**: musicals, theater, dance, ballet, jazz, classical, opera
- **Time filters**: This Week, This Weekend, Free Events
- **Sort**: by date, name, price (low/high)
- **Search**: full-text across title, venue, artist, description
- **Save events**: heart/bookmark with localStorage persistence
- **"Happening Now" indicators** for currently running shows
- **Event detail modal** with tickets link, Google Maps, .ics download
- **Sources panel** showing all 14 data sources with direct links
- **Keyboard navigation** and accessibility (ARIA, focus-visible)
- **Responsive** design for mobile, tablet, desktop
- **PWA manifest** for "Add to Home Screen"
